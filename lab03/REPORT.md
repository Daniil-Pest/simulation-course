### Клеточные автоматы. Лесные пожары (GUI)

**Дополнительные правила для поведения системы**:
* пустые ячейки (также конечные состояния) при генерации, не взаимодействующие с другими;
* направление ветра (wind_direction), усиливающее вероятность начала горения в полтора;
* влажность (humidity) — множитель для вероятность начала горения (от 0 до 1);
* задержка в остановке горения и рассеивании пепла — по rand().
